# 哈希表：把查找从线性扫描变成常数时间

## 1. 问题

给定数组 `nums` 和目标值 `target`，找出两个数，使它们的和等于 `target`，并返回它们的下标。
例如 `nums = [2, 7, 11, 15]`，`target = 9`，答案是 `[0, 1]`。

## 2. 暴力思路的问题

双重循环的时间复杂度是 O(n²)。当数组变大时，会非常慢。

## 3. 用 Map 换时间

我们真正需要知道的是：当前数字 `num` 的搭档 `target - num` 之前有没有出现过？
如果出现过，就直接返回两个下标。因此可以用 Map 保存 `value -> index`。

## 4. 一次遍历模板

```ts
export function twoSum(nums: number[], target: number): number[] {
  const seen = new Map<number, number>();
  for (let i = 0; i < nums.length; i++) {
    const complement = target - nums[i];
    if (seen.has(complement)) {
      return [seen.get(complement)!, i];
    }
    seen.set(nums[i], i);
  }
  throw new Error("No solution found");
}
```

## 5. 你要完成的练习

实现 `twoSum(nums: number[], target: number): number[]`。
要求：返回两个下标；假设每组输入都有唯一解；不要重复使用同一个元素。
