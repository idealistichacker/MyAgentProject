# JavaScript 基础表达式：从 1+1=2 开始

## 1. 什么是表达式？

在 JavaScript 中，**表达式**就是一段可以被计算并得到结果的代码。

例如：

```js
1 + 1
```

这行代码表示：把数字 `1` 和数字 `1` 相加。

它的结果是：

```js
2
```

所以我们可以说：

```js
1 + 1 === 2
```

这里 `===` 表示“严格等于”，意思是左右两边的值完全相同。

---

## 2. 数字

JavaScript 可以直接表示数字：

```js
1
2
10
100
```

这些都被称为**数字**。

在 JavaScript 中，整数和小数都可以使用数字类型表示，例如：

```js
5
3.14
```

本单元我们主要使用整数，例如 `1`、`2`、`3`。

---

## 3. 加法运算符 +

加号 `+` 是一个**运算符**。

当 `+` 放在两个数字之间时，它表示加法。

例如：

```js
1 + 1
```

表示：

```js
1 加上 1
```

结果是：

```js
2
```

再比如：

```js
2 + 3
```

结果是：

```js
5
```

```js
10 + 20
```

结果是：

```js
30
```

---

## 4. 输出结果：console.log

写好了表达式以后，我们通常想看到它的结果。

JavaScript 中常用：

```js
console.log(...)
```

来输出内容。

例如：

```js
console.log(1 + 1);
```

运行后会输出：

```js
2
```

这里的 `console.log` 可以理解为：

> 把括号里的内容显示出来。

---

## 5. 赋值表达式

有时候，我们希望把计算结果保存起来。

可以使用 `let` 声明一个变量：

```js
let result = 1 + 1;
```

这句话的意思是：

1. 先计算 `1 + 1`
2. 得到结果 `2`
3. 把结果保存到变量 `result` 中

然后可以输出这个变量：

```js
console.log(result);
```

输出结果是：

```js
2
```

完整代码：

```js
let result = 1 + 1;
console.log(result);
```

---

## 6. 从 1+1 改成别的数字

你已经知道：

```js
console.log(1 + 1);
```

会输出：

```js
2
```

现在可以试着改成：

```js
console.log(2 + 3);
```

它会输出：

```js
5
```

也可以改成：

```js
console.log(10 + 20);
```

它会输出：

```js
30
```

这就是编程中非常重要的一步：

> 写代码，运行代码，观察结果，然后修改代码再试一次。

---

## 7. 小结

你已经学习了 JavaScript 中非常基础的内容：

- 数字：例如 `1`、`2`、`3`
- 加法运算符：`+`
- 表达式：例如 `1 + 1`
- 输出：`console.log(1 + 1);`
- 变量：`let result = 1 + 1;`

最重要的一句话是：

```js
console.log(1 + 1);
```

它会输出：

```js
2
```
