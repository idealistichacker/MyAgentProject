# 栈与括号匹配：用 LIFO 表达嵌套结构

## 1. 什么是栈？

栈是一种后进先出的数据结构，英文叫 Last In First Out，简称 LIFO。
常见操作：`push` 入栈，`pop` 出栈，`peek` 查看栈顶。

## 2. 括号匹配为什么需要栈？

括号具有嵌套关系，例如 `([{}])`。最右边的左括号 `{` 必须最先被匹配，这正是栈的特性。
当你遇到右括号时，它必须匹配最近一个还没匹配的左括号。

## 3. 判断规则

1. 遇到左括号，入栈
2. 遇到右括号：如果栈为空则失败；弹出栈顶；检查是否匹配
3. 遍历结束后，栈必须为空

## 4. 模板

```ts
export function isValidParentheses(s: string): boolean {
  const stack: string[] = [];
  const pairs: Record<string, string> = { ')': '(', ']': '[', '}': '{' };
  for (const ch of s) {
    if (ch === '(' || ch === '[' || ch === '{') {
      stack.push(ch);
    } else {
      const top = stack.pop();
      if (top !== pairs[ch]) {
        return false;
      }
    }
  }
  return stack.length === 0;
}
```
